Nameplates

Upload a class roster (a Canvas gradebook export or a plain name list) and print tent-fold desk nameplates, three to a page. Each name sits low in its third with blank space above. Cut the page into strips and fold each one so the name stands up on the desk. Check the parsed first names below, fix any the tool got wrong, then print to PDF.

Class roster
Canvas export: drop it in as-is. The tool finds the Student column and skips the points and test rows.
Plain list: one name per line, or a CSV with names in the first column.
How the first name is read. For a Canvas "Last, First" entry the tool takes the first given name after the comma. For a "First Last" entry it takes the first token. It keeps a hyphen (Mary-Jane), drops middle names (John Michael becomes John), and leaves surname particles where they belong (van, de, del, bin, al- stay with the last name). A single-token name (Madonna) stands on its own. If the roster has a nickname or preferred-name column, that wins. Names are guesses, so read the list below and fix anything wrong before printing.
